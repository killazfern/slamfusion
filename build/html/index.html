Wrapper method that calls the various methods to create the merged map.

Parameters:
  • varx (Integer.) – Start value for coordinate X.
  • vary (Integer.) – Start value for coordinate Y.
  • step (Integer.) – Step value for the the cycles.
  • prestep (Integer.) – Step value for the previous call of the method, At start is 0.
  • trigger (Integer.) – A trigger value, at first run it is 0, following runs it is 1.
  • firstmap (Float matrix.) – Contains the map values, from one of the bigger maps, usualy Gmapping.
  • secmap (Float matrix.) – Contains the map values, from the smaller map, usualy Karto.
  • soma (Tuple array with values (Error value,Start coordinate X,Start coordinate Y,Rotation).) – The array where the calculations are saved.
  • rot (Integer.) – Rotation value.
Returns:

The original Tuple array with the new calculations.
